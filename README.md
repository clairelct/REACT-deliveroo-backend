## Running the project

Clone this repository :

```
git clone https://github.com/clairelct/REACT-deliveroo-backend.git
cd REACT-deliveroo-backend
```

Install packages :

```
npm i
```

When installation is complete, run it :

```
npx nodemon index.js
```
